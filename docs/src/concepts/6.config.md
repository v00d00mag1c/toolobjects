### Config

`Config` is a singletone that contains options, values etc. Config files are placed in `src/config` dir. They are divided on `config.json` - values, settings and `env.json` - sensitive information.

As Executables, it uses [Arguments](../objects/App.Objects.Argument.md) to define and validate values. The settings of each class are defined in the `_settings` function:

```
class Reader(Object):
    @classmethod
    def _settings(cls):
        return [
            Argument(
                name = "books.reader.show_oage_number",
                orig = Boolean,
                default = True,
                role = "config"
            )
        ]
```

`role` defines state of the option (`env` of `config`).
