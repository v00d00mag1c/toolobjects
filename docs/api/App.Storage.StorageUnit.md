### App.Storage.StorageUnit

File or many files in separated directory, must contain common file

`hash`: randomly-generated hash (not relies on file)

`files`: list of [Media.Files.File](Media.Files.File.md)

`name`: name of common file, `ext`: extension

`common`: relative path to common file (`setCommonFile()`)
